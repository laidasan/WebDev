序言

這一系列的日記屬於Javascript日記，首先先要講這系列日記較偏向紀錄Javascript語言的特性、觀念與需要注意的的地方，比如說ES6中最新的let與傳統的var的差別，或是"變量提升"的問題。

在寫此序言的此時此刻的我，最一開始學語言是從Java語言開始的，Java與javascript是兩種不同的語言，也有很大的差別在(特性上的差別)，最大的差別在於Java是屬於強型別語言、靜態語言;javascript是屬於弱型別語言、動態語言， 但是並沒有說哪個語言是最好的，每個語言都像是人一樣，各有優缺點，語言規則也是人類設計出來的，通常都是環境、科技進步程度去影響，但同樣的都還是會有個"為了解決某些為題"的目標，而各有不同的目的與要解決的問題，才有不同的特性。

就拿屬於強型別的語言"Java"來說，Java屬於靜態語言 ，它的規則較多也較嚴謹，就拿變數的宣告來說，Java變數宣告一定要告知是哪種類型的資料，如果是String就要告知是String，而且之後再指定、給予值的時候就一定要是String，否則會編譯會不給過!

而javascript來說，變數宣告時可以不用表明是哪種類型的資料，javascript屬於動態語言 ，它的資料類型是依照我們給的值去做決定 ，比如說今天給的值是number，此變數資料類型就是number，且之後可以給予、指定任何類型的值，資料類型也會跟著改變。

※強型別;弱型別只是個形容詞，原本是沒具任何意義的，是我們給予意義的，所以重要的點是看Java or javascript etc..這些語言設計上是如何去設計的。

每種語言就像人一樣都會擁有自己的特型(目前都是人設計出來的嘛!)，對於我來說，沒有最好的語言，只有更好、與對於解決問題，針對不同目的與目標更適合、更好的語言，且要牢牢記住:

語言、技術只是工具，別被工具牢牢銬住了思想、思考跟價值觀，最重要的還是能夠發揮出價值，幫助他人。
