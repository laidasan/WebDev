My Note

DataType

What?

今天來講講在Javascript裡面，資料的型態有哪些，為何要了解資料的型態呢?其實電腦或是網頁等等說白了就是處理一堆資料後在視覺話出來，而我們也必須告訴電腦資料的型態與定義讓他們好去處理吧! 就像我們人類在溝通一樣，我們知道什麼是數字，進而以做數子的相加，我們知道是句子，我們才能夠將句子做變化、造樣造句等等。

How?

主要分成兩種型態

  • 基本型態--primitaive Data Type
  • 複合型態--Reference Type

還有幾種特殊值

  • null
  • undefined
  • NaN

基本型態--primitive data type有下列幾種

  • string--字串
  • number--數字
  • boolean--布林值

number

在Javascript裡面沒有整數與浮點數的分別，通通都是number，數值都是IEEE 754標準64位元浮點數，可表示的數字最小值為Number.MIN_VALUE，最大為Number.MAX_VALUE。

跟其他程式語言一樣，都是用實字來表示，預設是十進位整數，也可以用"0"開頭來表示八進位，"0x"開頭來表示16進位。

let num1 = 10;

let num2 = 010;

let num3 = 0x10

如果要使用浮點數可以直接寫上，或是用科學記號

let float1 = 3.12

let float2 = 5.312E13

let float2 = 5.312E-13

數值也有幾個特殊值，分別為無限大"+Infinity";負無限大"-Infinity"，也可以使用"Number.POSITIVE_INFINITY;Number.NEGATIVE_INFINITY"來取得。

還有一個是NaN--Not a Number，就跟字面上意思一樣，不是數字，這個很常出現在運算的時候，比如說我們將1除上字串就會得到NaN

let num = 1 / 'two';

let nan = NaN;

對於NaN有些事情需要注意，NaN 不等於 NaN，但是可以使用 isNaN來檢查是不是NaN

let nan = NaN;

nan == NaN//false

isNaN(nan) //ture

string

string也是基本型態之一，可以使用雙引號或是單引號來宣告，在Javascript裡沒有字元的型態(char),所以用單引號刮上一個字的話，型態也還是string

let str1 = 'joe';

let str2 = '';

typeof str1 //string

typeof str2 //string

"typeof"可以用來檢測資料的型態，比如說位數值始就會回傳number，字串就會回傳string。

通常我們在使用上會使用單引號比較多，會把雙引號留給Html使用(像是class=" ")。

boolean

boolean就比較簡單了，就只有true與false兩種，但之後會提到其實在javascript裡面所有的型態都可以轉換成boolean。

複合型態

  • Object literal
  • Object(實例)
  • Array
  • function
  • Data
  • etc....

這裡會簡單的提到，而function與Array之後會在細談

let obj = new Object();

let person = { }

let ary = [1,2,3];

let date = new Date() //Date是時間的API`

以上這些typeof都會回傳object喔!除了typeof之外還可以使用"instanceof"來檢測，更詳細的instanceof是用來檢測constructor.prototype是否存在於某個原型鍊上(之後會再詳細談談prototype)，簡單來說"instanceof"可以用來檢測物件是誰的實例

let ary = [1,2,3];

ary instanceof Array //true

ary instanceof Object //true

接下來我們會談談在javascript裡面特殊的兩個值，null v.s undefined。

null

null表示沒有任何東西

let nul = null;

nul == null //true

typeof nul;//Object

nul instanceof Object //false

※這一行很重要，null有一個bug，就是用typeof的話會回傳Object，但實際上null不是Object喔!!!

undefined

undefined會在我們宣告變數卻沒有給值的時候，或是function沒有return時etc....。

let x;

function note( ) { }

console.log(x); //undefined

console.log(note()); //undefined

※這一行很重要，假如是 "let x = null"，x參考一個值null，所有x不是undefined喔!!
