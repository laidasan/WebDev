My Note

型態轉換

What?

在前一篇裡面我們有講到了DataType(資料型態)的部分，可以知道主要分成兩種 "Primitive type"(基本型態)與"Reference type(複合型態)"，接下來我們來介紹不同的形態之間在做運算或是型態轉換(比如說string轉int)時有哪些需要注意的地方。

How?

在像是Java這種強型別語言裡面，對於型態是非常嚴格的，比如說在Java裡面將字串與數字相減的時候編譯就會出錯

String str = "1";

int i = 1;

System.out.println(str - i);

要能夠正常相減的話就必須將String轉換成int才行

String str = "1";

int i = 1;

System.out.println(Integer.parseInt(str) -i);

如果是在javascript裡面就可以直接做(會自動轉換型態)

let str = '1';

let i = 1;

console.log(str - i);

console.log(typeof (str - i) );

我們也可以用typeof來看看轉換型態後是什麼型態(number)。

javascript的基本型態number,string,boolean會在必要的時候，自動型態轉換為對應的包裹物件Number,Stgring,Boolean。

let num = 10;

num.toString();

(10).toString();

num指定變數10型態是number，但是在操作toString是使用物件上的方法(number本身沒有)，這個時候就會自動使用Number實例來包裹number10，因此才能操作toString()這個動作，實字(10)亦相同。

類似的在string上面也會有這種情況發生。

let text = 'joe';

text.toUpperCase();

'joe'.toUpperCase();

同樣的toUpperCase()方法只存在物件上面，string基本型態上是沒有的，這時候就會用String實例來包裹string型態，才可以操作toUpperCase()

或是我們可以自己創建實例

let num = new Number(10);

let text = new String('joe');

以上兩個num,text使用typeof出來的話都是Object歐!!至於使用function實例出來的這種方式我們留到以後再說。至於更多的Number,String,Boolean物件的方法可以去查詢API!!

我們也可以使用parseInt()、parseFloat()將字串轉為數值，好處是會自動忽略字串尾端非數字部分。

parseInt('10 years old');

parseInt('3.14159.....')

parseInt('010',10);

parseInt('010',8);

好玩的是parseInt可以指定要使用10進位或是8進位。

運算

型態的轉換很常在需要運算的時候(不管是自動轉換或是我們手動轉換)，運算就包括上了+,-,*,/，等等會介紹下當不同型態再做加減乘除的時候

let str1 = '6'

let str2 = '3'

str1 + str2

str1 - str2

str1 * str2

str1 / str2

可以試著實做看看上面的結果為如何。當為加法的時候會優先做字串相加，所以str1 + str2的結果是62(string)，減乘除的時候才做數字上的運算

也可以和boolean做運算

1 + true;

1 + false;

在做+,-,*,/的運算的時候，true會被當作1，false會被當作0

有一個很有趣的是，在javascript裡，任何東西都可以變成boolean值，0,NaN,null,undefined,''，以上這些轉換成boolean時都會轉換成false，其他都是true,所以他們有個稱呼叫做"FalseFamily"

可以運用這個小技巧，若某物件上不存在某個特性，直接取用該特性的話會回傳undefined，爾後我們可以利用undefined轉換成false的原理來知道該物件上有沒有某個特性，比如說

const person = {

name: 'Joe',

age: 22

}

if(person.age) {

console.log('have')

}else {

console.log('do not have')

}

因為person上是有age特性的，所以回傳不會是undefined，當轉換成boolean的時候會轉換成true，所以if裡面的"console.log('have')就會觸發囉!!

我們還可以使用==來比較兩邊是不是相等，比如說

let str1 = 'joe'

let str2 = 'joe'

let num = 3

let strn = '3'

str1 == str2

num == strn

num === strn

str1跟str2的比較簡單，兩個都是字串也都是joe，理所當然是回傳true囉!但是可以注意一下，在javascript裡面除了==之外，還有===三等號喔!三等號事完全等於的意思，除了值要相同，型態也需要相同才會回傳true喔! "num == strn"會回傳true，"num === strn"會回傳false，其原理是只用==來做比較的話，如果兩邊型態不同，javascript會幫我們嘗試轉換成相同的型態看看，試試看是否會相等，而===三等號就不會了，就一定要兩邊型態與值都相同才會回傳true囉!!

至於物件的以較，我們留到下一篇再說囉!
