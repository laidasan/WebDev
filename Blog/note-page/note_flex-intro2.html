Flex-詳介

What? For?

上篇已經很簡單的使用flex,也感受到他的方便了，這篇就來詳詳細細的介紹它的原理以及更多的介紹吧!!

How?

Flex最最最重要就三個大重點:

  • container & item (容器與內容器;父層與子層)
  • Flexbox module (主軸與副軸)
  • flex-grow;flex-shrink;flex-basis

Flexbox module

我們先來看看Flexbox module吧!其實他相當簡單!下面這張圖就是flexbox module。

我們先來看 main axis 與 cross axis， main axis就是主軸的意思，而cross axis就是副軸，那什麼是主軸?什麼是副軸呢?我們最一開始我們可能會想:恩?阿不就是X軸跟Y軸嗎? 其實不是喔!雖然現在看到圖片上的主軸跟副軸可能跟X軸Y軸的方向一樣，但是!!flex的主軸與副軸不是這樣分的。

主軸指的是當前flex排列的方向 ; 副軸指相對於主軸的縱向軸。

像圖片上看到的主軸為橫向的，那副軸就是縱向，而我們可以去更改排列的方式[row(橫、行) or col(縱、列)]，如果這時候排列的方像是縱向的，那主軸就是縱向，而副軸就是橫向， 這非常非常重要喔!要記起來!!

Container $ item

接下來就可以講講屬性了，在講屬性之前，我們先把屬性分類一下，大致上分為兩種，一種是給container的，也就是父層容器，一種是給item的，也就是父層容器內的內容物，再父層容器設置的大多是將item如何排列， 例如排列的方向為橫向或是縱向、是否要換行、主軸與副軸的對其方式，而內容的設置大多是當父層寬度伸縮時item自己本身要如何伸展等等!

OK!接下來就可以來講講屬性了!!

  • flex-start (Default)
  • flex-end
  • center
  • space-between
  • space-around

flex-basis

flex-basis跟width很像，但它不是指定寬度，而是主軸方向的長度，比如說今天我的排列方式(flex-decoration)是row，那flex-basis就會是寬度(左右);反之如果主軸排列方式為col，那flex-basis就是指定高度了。

※如果同時設定了width與flex-basis，會以basis優先喔!!
