My Note

Reset

What?

「CSS reset」就是先把所有標籤的css做一個設定(比如說margin、padding等等)，為什麼要這麼做呢?請聽我娓娓道來。

身為網頁工程師的我們，我們的好朋友之一就是瀏覽器拉!!!我們寫出來的原始碼就是給好朋友們看得(google、IE、firefox etc...) 每個人都是獨立的個體，有不同的想法，不同的作為與方式，這是很正常的，以這個觀點來看瀏覽器也是一樣的，對於瀏覽器來說，不同的瀏覽器也有不同的"css 預設值"，就像是每個瀏覽器都有不同的個性一樣， 也許我們會有個問題，如果沒有這些瀏覽器預設值的話會怎麼樣啊?「這樣如果我們寫網頁的時候沒有設給CSS，不就什麼都看不到了嗎?!!!」

也因為每個瀏覽器都有不一樣的預設值，所以當我們在寫的網頁用不同瀏覽器開啟的話，可能會有些許的差異，這時我們就會很難地去掌控所有的瀏覽器顯示的網頁效果都是我想要的， 而為了解決這個問題，我們就想到了，如果再寫我們自己的CSS的時候，先把所有的標籤設定都先設定一個"統一"的標準，這樣我們就能容易的掌控各家的瀏覽器顯示出來的效果啦!!!!CSS reset真是太棒了!!

How?

再這裡說說兩個CSS Reset， CSSRESET 與 Normalize ，這兩個是現在很多人常用的CSS reset，css reset做的reset比較誇張，但是更能夠掌控網頁我們想要的樣子，而Normalize沒有css reset那麼誇張(筆者再用的是css reset)， 而使用的方式呢也很簡單，以CSS RESET來說我們通常都會再建立一個檔案叫做reset.css，之後我們在我們想要使用之html "head"裡面導入就好，就像是導入我們自己的css一樣。

※導入時要注意先後順序喔!要把css reset寫在最前面，在讀取原始碼的時候是一行一行往下讀的。
