My Note

Flex

What?

Flex是種排版的方式，非常異常的方便，自己本身覺得它是排版的聖品，也是排版的毒品，用過就回不去了? 今天就來使用與介紹下Flex吧!!

How?

我們先來看看一個flex的例子。 ←Codepen

可以去試著去增加item(wrap裡面的div)，你會發現!!!唉唷!!不會壞掉耶!如果是傳統的float排版，只要寬度沒有計算好就會壞掉(換行)了! 可以再試著去把某個item的字數減少，看看回不會壞掉，你會再次發現!唉唷!!!! 也不會壞掉餒!!也太神奇了吧 不會因為沒有計算好，就造成換行的現象，但這時候我們可能會想:嗯?這樣我要換行的話要怎麼辦啊?!

Flex有一個屬性叫做"flex-wrap可以去設置，這個屬性就是設定在一行寬度容不下的時候，就會自動換行， 我們寫上"flex-wrap: wrap"這樣就可以自動換行囉!! ←Codepen

在預設的情況下，item的高度會是等高的(高度會被伸展開來，其他筆記會有更詳細的介紹)。 flex實在是很方便快速，可以很快速就可以完成一些很簡單排版，比如說 ←Codepen 很簡單的只要一個flex，而且可以不用去計算width，就可以快速又簡單的排簡單的版拉!!(但也並不推薦把width亂設拉!還是會出問題的)
