My Note

Box-module new box

What?

之前我們有談到box-module，在基礎上(預設上)，我們一個區塊的尺寸是padding + 內文寬(長)度 + 邊框。 像下面這個div我們寬度設定為300，padding設定為0 12px,border設定為9px

我們有一個屬性叫做"box-sizing"，這個屬性跟我們的box-module有什麼關係阿?!!

How?

box-sizing簡直是一個劃時代，超方便的屬性，以往所有的瀏覽器預設情況都是像上面的計算方式計算出來的，box-sizing的方便性就是 我們可以設定成border-box，看看發生什麼事情。

會發現寬度變小了，且是剛好我們設定的300px，非常非常之方便阿有木有，我們就不用再去計算尺寸拉! 這個的原理是什麼呢?其實box-sizing是去控制我們盒模型的計算方式 ，簡單來說box-sizing控制我們的寬度值是要哪個盒子， 像我們設定成border-box的意思就是"把我們的寬度設定給border-box(邊框到邊框的寬度) ，如果我們設定成"content-box"就會變成原本計算的方式了，content-box意思就是寬度設定給"內容物寬度"，所以padding與border就會往外加。

有了box-sizing我們可以更偷懶......不是，我是說可以更方便的去做排版，假如說今天我們做了個三欄式的排版，以往都需要去精精計算尺寸，只要稍微的多了幾個像素 就會跑版，那有了box-sizing就可以減少計算(margin還是往外加的，所以要去計算margin)，更快速方便的排版囉!!比如說之前做的三欄式排版 ←Codepen
